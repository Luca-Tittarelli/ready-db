# ReadyDB

ReadyDB is a premium, warm-minimalist tool for designing, visualizing, and exporting relational database schemas directly in your browser. Built with a focus on speed, clarity, and modern aesthetics.

## Features

- **Visual Schema Design:** Drag, drop, and connect tables on an infinite canvas.
- **Smart Type Selection:** Professional UI separating base types (VARCHAR, INT, etc.) from arguments/lengths.
- **Relational Connections:** Intuitive 1:1, 1:n, n:1, and n:n foreign key connectors with classic "crow's foot" notation.
- **Zero Dependencies:** All state is persisted automatically to your browser's local storage.
- **SQL Generation:** Export your visual schema instantly to valid, production-ready SQL (PostgreSQL, MySQL).
- **JSON Export/Import:** Backup and share your schema layouts via JSON files.
- **Warm Minimalist UI:** Clutter-free interface, highly functional, designed with a focus on a "warm Apple-like" aesthetic.

## Tech Stack

- React + Vite
- Tailwind CSS v4
- Lucide React (Icons)
- upop (Modern Alerts)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/readydb.git
   cd readydb
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.
